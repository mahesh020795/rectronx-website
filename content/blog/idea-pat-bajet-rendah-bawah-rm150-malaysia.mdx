# 20 Idea Projek Akhir Tahun Bajet Rendah Bawah RM150 untuk Pelajar Malaysia

Tak semua pelajar ada bajet RM300-500 untuk projek akhir tahun, terutamanya pelajar politeknik atau diploma yang selalunya kena tanggung kos sendiri. Berita baik: bajet rendah tak bermakna projek kau kena nampak murahan. Senarai 20 idea ni semua boleh siap dengan kos komponen bawah RM150, tapi tetap ada substance teknikal yang cukup untuk isi laporan dan jawab soalan panel dengan yakin.

---

## Berasaskan Sensor Tunggal (Paling Murah)

**1. Sistem Amaran Suhu Bilik Menggunakan DHT22** — ESP32 + DHT22 + buzzer, amaran bila suhu melebihi julat selesa. Kos anggaran RM 40-55.

**2. Penunjuk Aras Air Tangki Mudah** — Sensor ultrasonik HC-SR04 + LED/buzzer untuk tunjuk aras air dalam tangki kecil. Kos anggaran RM 30-45.

**3. Sistem Pengesan Gerakan Ringkas Menggunakan PIR** — Sensor PIR + buzzer/LED untuk pengesanan kehadiran di ruang kecil, asas keselamatan rumah. Kos anggaran RM 25-40.

**4. [Sistem Pengesanan Gas LPG Asas](/blog/sistem-pengesanan-gas-lpg-kebakaran)** — MQ-2 + buzzer + LED, versi ringkas tanpa kawalan injap automatik. Kos anggaran RM 35-50.

**5. Alat Ukur Kelembapan Tanah untuk Pasu Bunga** — Sensor kelembapan tanah + LED penunjuk (merah/hijau), tak perlukan servo atau pam automatik. Kos anggaran RM 25-40.

---

## Projek Berasaskan Arduino Uno Klon (Sangat Murah)

**6. Lampu Trafik Simulasi Menggunakan Arduino** — Arduino Uno klon + LED merah/kuning/hijau + logik pemasaan, projek klasik yang senang dijelaskan tapi tetap tunjukkan logik pengaturcaraan. Kos anggaran RM 30-45.

**7. Sistem Kunci Digital Menggunakan Keypad 4x4** — Arduino + keypad membran + servo sebagai kunci solenoid simulasi. Kos anggaran RM 45-65.

**8. Robot Pengesan Halangan Asas** — Arduino Uno klon + satu sensor ultrasonik + motor DC + chassis murah. Kos anggaran RM 60-85.

**9. Jam Loceng Elektronik dengan Paparan LCD** — Arduino + modul RTC DS3231 + LCD 16x2 + buzzer, projek elektronik digital asas yang kukuh dari segi konsep. Kos anggaran RM 40-55.

**10. Alat Kira Denyutan Jantung Ringkas** — Sensor denyutan jantung mudah (pulse sensor) + Arduino + paparan LCD, projek kesihatan asas dengan kos rendah. Kos anggaran RM 45-60.

---

## Berasaskan Perisian/Aplikasi (Kos Komponen Minimum)

**11. Aplikasi Pengurusan Kehadiran Kelas Mudah** — Aplikasi web/mudah alih asas untuk rekod kehadiran pelajar, tiada kos hardware — hanya kos hosting/domain minimum jika perlu. Kos anggaran RM 0-30.

**12. Sistem Pengurusan Inventori Kedai Kecil** — Aplikasi desktop/web untuk urus stok dan jualan perniagaan kecil, sesuai untuk pelajar Sains Komputer atau IT. Kos anggaran RM 0-20.

**13. Chatbot FAQ Ringkas untuk Laman Web Universiti** — Bina chatbot asas menggunakan rule-based logic atau API percuma untuk jawab soalan lazim pelajar. Kos anggaran RM 0-30.

**14. Aplikasi Kalkulator Kewangan Peribadi untuk Pelajar** — Aplikasi mudah alih untuk jejak perbelanjaan dan belanjawan bulanan pelajar. Kos anggaran RM 0-20.

**15. Portal Semakan Gred Pelajar Ringkas** — Sistem web asas untuk pelajar semak gred dan progress akademik, sesuai untuk projek FYP berasaskan pangkalan data. Kos anggaran RM 0-30.

---

## Gabungan Sensor Kos Rendah

**16. Sistem Pemantauan Suhu dan Kelembapan Stor Ringkas** — DHT22 + ESP32 + paparan OLED (tanpa GSM/SMS), fokus kepada pemantauan tempatan sahaja. Kos anggaran RM 55-75.

**17. Alat Pengesan Kualiti Udara Asas Menggunakan MQ-135** — Sensor MQ-135 + ESP32 + buzzer amaran, versi ringkas tanpa dashboard atau logging jangka panjang. Kos anggaran RM 50-70.

**18. Sistem Kawalan Lampu Bilik Menggunakan Sensor Cahaya (LDR)** — LDR + Arduino + relay untuk kawal lampu automatik ikut tahap cahaya persekitaran. Kos anggaran RM 40-60.

**19. Alat Pengesan Hujan Ringkas untuk Amaran Basikal/Pakaian Terjemur** — Sensor hujan + buzzer/LED + ESP32, projek mudah dengan aplikasi harian yang jelas. Kos anggaran RM 35-50.

**20. Sistem Penunjuk Bilangan Tempat Duduk Kosong Ringkas** — Sensor ultrasonik di beberapa tempat duduk simulasi + paparan LED/LCD status kosong/penuh, versi kos rendah untuk konsep [sistem parking pintar](/blog/smart-parking-system-fyp-guide). Kos anggaran RM 60-85.

## Cara Jimat Lagi Tanpa Jejaskan Kualiti Projek

Beli komponen dalam kuantiti (bundle set sensor) daripada platform seperti Shopee/Lazada biasanya lebih murah berbanding beli satu-satu daripada kedai elektronik fizikal. Guna Arduino Uno klon (bukan yang asli) untuk projek yang tak perlukan jenama tertentu — perbezaan fungsi biasanya tak ketara untuk keperluan FYP. Kalau universiti kau ada makmal elektronik dengan komponen sedia ada untuk pinjam, semak dulu sebelum beli sendiri — ini boleh jimat sehingga separuh bajet kau.
