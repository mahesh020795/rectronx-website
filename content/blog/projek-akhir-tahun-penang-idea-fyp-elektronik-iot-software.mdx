## Quick Summary

Untuk pelajar Penang, tajuk yang kuat biasanya berkaitan elektronik, IoT, software, automation, dashboard atau mobile app. Pilih projek yang mempunyai demo jelas, bukan tajuk terlalu besar yang susah siap.

Jika anda mahu bantuan tempatan, lihat [FYP Project Support Penang](/services/fyp-project-penang).

## Projek Elektronik dan IoT

Untuk elektronik dan IoT, komponen yang biasa digunakan:

- [ESP32](/components/esp32)
- [Arduino Uno](/components/arduino-uno)
- [RC522 RFID](/components/rc522)
- [BME280](/components/bme280)
- [DHT22](/components/dht22)
- [Relay Module](/components/relay-module)

ESP32 sesuai jika projek perlu WiFi, dashboard atau app. Arduino sesuai jika projek offline dan lebih mudah.

## Projek Software dan App

Untuk software, pilih sistem yang mempunyai user flow jelas:

- login
- create record
- admin review
- update status
- report atau dashboard

Contoh yang sesuai ialah sistem aduan pelajar, sistem tempahan bilik, sistem inventory makmal atau sistem kehadiran.

## Cara Pilih Skop

Untuk elak projek menjadi terlalu besar:

1. Pilih satu masalah utama.
2. Pilih satu jenis pengguna utama.
3. Tentukan output demo.
4. Senaraikan komponen atau software stack.
5. Tetapkan basic version dahulu.

Jika basic version sudah stabil, barulah tambah dashboard, notification atau casing.

## Kesilapan Biasa

- Pilih tajuk terlalu besar kerana mahu nampak hebat
- Beli komponen sebelum semak skop
- Tiada backup demo jika WiFi gagal
- Tiada testing evidence
- Laporan tidak selari dengan prototype
