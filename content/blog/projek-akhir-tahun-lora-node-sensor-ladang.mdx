## Ringkasan Cepat

Projek LoRa sesuai untuk pelajar yang mahu IoT jarak jauh tanpa bergantung sepenuhnya kepada WiFi. Sistem ini menghantar bacaan kelembapan tanah, suhu dan bateri daripada node sensor kepada gateway.

## Skop Projek

Skop realistik ialah satu gateway dan dua atau lebih node sensor. Setiap node membaca sensor, menghantar data secara berkala, dan gateway simpan atau paparkan data.

## Komponen Dicadangkan

Gunakan ESP32 atau Arduino, modul LoRa SX1276 atau RA-02, sensor kelembapan tanah kapasitif, DHT22 atau BME280, bateri, regulator, dan dashboard.

## Langkah Bina

1. Uji satu pasangan transmitter dan receiver LoRa.
2. Masukkan ID node dalam data.
3. Tambah bacaan sensor.
4. Simpan data di gateway.
5. Jalankan ujian jarak dan rekod kadar data berjaya diterima.

## Nota Ketepatan

Bacaan kelembapan tanah perlu dikalibrasi menggunakan sampel tanah kering dan basah. Jarak LoRa bergantung kepada antena, halangan, ketinggian dan tetapan frekuensi.

## Kesilapan Biasa

- Tiada ID node dalam paket.
- Antena diletak terlalu rendah.
- Hantar data terlalu kerap tanpa sebab.
- Anggap bacaan mentah sensor sebagai peratus tepat.
- Tiada pemantauan voltan bateri.
